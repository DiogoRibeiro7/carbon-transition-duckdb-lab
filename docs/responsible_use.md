# Responsible Use

This repository is for exploratory analysis.

Do not use the score as a moral ranking or as a final policy judgement. Country
energy systems are shaped by many constraints, including geography, income,
industrial structure, infrastructure, resource availability, and historical
development paths.

The score is useful for asking better questions:

- Which countries have rising emissions despite energy transition?
- Which countries have high fossil dependence?
- Which countries combine high carbon intensity and low renewable electricity?
- Which rows need deeper investigation?

The project should not be used to shame countries or make unsupported causal
claims about policy success or failure.
